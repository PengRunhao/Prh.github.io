使用云服务器训练模型

本文用来记录我是如何租用AutoDL的云服务器来训练自己的深度学习模型的。 1.登录AutoDL官网，在算力市场租用一台GPU，在基础镜像栏选择自己使用的框架与版本，如pytorch1.7.0+python3.8+cuda11.0 2.租好GPU之后，AutoDL的控制台会显示当前GPU的登录指令与密码 3.在PyCharm(专业版)中添加该GPU的环境依次点击File => Settings => Project => Python Interpreter 点击Add后， 填完之后点击Next， 填完点击Next， 完成后点击Finish， 现在我们成功将PyCharm配置完成 4.将本地文件上传到服务器上打开Xshell，连接服务器 之后会弹出对话框， 点击确定后出现以下页面，则说明连接成功 接下来上传文件到服务器，如果文件过大，可以压缩后再传输 5.在服务器端查看 运行结束后可以将文件再打包通过上面的步骤发送到本地。 至此一套完整的用云服务器训练自己模型的流程就完全完成了！
