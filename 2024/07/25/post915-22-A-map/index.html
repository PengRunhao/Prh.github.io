西交22年代码第一题：无人机打靶

题目描述：

有 N 支队伍参赛，每个队伍的无人机成绩为 S 且每个无人机颜色被随机涂为蓝（b）、红（r）、绿（g）、紫（p）之一。

输入格式：

第一行包含整数N，表示参加的队伍数目，即无人机数目。

接下来 N 行，每行包含每支参赛队伍的无人机颜色和打靶成绩 S 由空格分隔开。

输出格式：

输出内容由两部分组成。
第一部分输出各种颜色对应的无人机数目。
第二部分根据按照成绩从大到小进行排序，输出无人机打靶成绩 S 和对应的无人机颜色。

数据范围：

1 <= N <= 30

0 <= S <= 100
